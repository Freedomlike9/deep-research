# Consulting Analysis

## Core Expectations

- Define a clear analytical objective.
- Use explicit chapter structure.
- Connect evidence to implications and recommendations.
- Prefer consulting-style phrasing over loose summaries.

## Recommended Structure

1. Executive Summary
2. Key Findings
3. Supporting Evidence and Comparisons
4. Risks and Counterpoints
5. Recommendations or Outlook
