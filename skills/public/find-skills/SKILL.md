# Find Skills

## When To Use

- The user asks whether a capability exists as a skill.
- The user wants to extend the agent with a new workflow.
- The task is more about finding reusable capabilities than solving the task directly.

## Working Style

1. Identify the missing capability clearly.
2. Map it to an existing skill if possible.
3. Recommend the smallest relevant skill set.
4. If no fit exists, suggest migration or creation of a new skill.
